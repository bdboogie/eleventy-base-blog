Create a tip calculator in javascript (Lesson 9 - Programming Concepts)

Your "Eat Out to Help Out" bill comes to £49.56,
if you would like to add a gratuity press the button below:

The final coded used to calculate the gratuity (tip) is listed below you can review chg11-tipCalculator.js for all the code iterations.

        
        function calculateTip() {
          var billAmount = 49.56;
          var tipPercent = 17;
          var tipAmount = (billAmount * tipPercent / 100);
          document.getElementById('totalBillMessage').innerHTML = 
          `Your total bill, including a gratuity of £${tipAmount.toFixed(2)}, comes to £${(billAmount + tipAmount).toFixed(2)}`;
      };
